# chat

# フロントエンド

**Vue**を使用する．UIはVuestic-UIを使用．
[こちら](https://ui.vuestic.dev/)を参照すること．

## Project setup

```bash
npm install
```

## .env
```
VUE_APP_API_KEY=''
VUE_APP_BASE_URL=http://127.0.0.1:8000/api/
VUE_APP_STATIC_URL=http://127.0.0.1:8000/static/
VUE_APP_WS_URL=ws://127.0.0.1:8001/
```
